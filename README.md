## Descrição
Projeto feito em Unity com C#, usando um único cubo na cena e um script denominado 'EcoQuest01Cube.cs'.
O projeto pratica conceitos básoicos de programação , como variáveis, operadores, condições, entrada de teclado e os métodos 'Start()' e 'update()'.

____________

## Funcionalidades 
O script implementa:

Mensagem de boas vindas;
Cálculo de pontuação;
Status inicial do herói;
Sistema de dano e vida;
Divisão de sementes entre canteiros;
Sistema de Game Over;
Verificação de chave de laborátorio;
Controle de velocidade;
Sistema de levelUp;
Escudo de Energia;
Coleta de lixo espacial;
Gasto continuo de energia;
Pulo seguro;
Rada de inimigos;
Sistema simples de pausa.
